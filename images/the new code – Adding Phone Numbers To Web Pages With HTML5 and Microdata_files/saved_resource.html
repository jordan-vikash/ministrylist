                                                                                                                                    Yves Florack Dan • 6 years ago

                                                                                                                                    In the case that Android devices or any browsers do this, using "00" will work in most countries, but definitely wouldn't cover the countries I would personally like to target. Another commonly used dial-out/country-exit code is "011", which is also used by the US, Canada and many Caribbean Countries. In a few South-American and Asian countries it's provider-dependent, and in some countries like Russia, Turkmenistan, and Kazakhstan an 8 is followed by a w(ait for dialtone) before entering 10 to dial-out. So in that case maybe a php or js script to detect current location would be needed (presuming the person isn't using a DNS or VPN connection). So, in brief, if you are targeting the International market, I would strongly argue against using "00" and hope that the discarding of the plus sign (+) is only on some Android devices or browsers, or hope they fix this issue soon. [see howtocallabroad.com for dial-out codes.]
